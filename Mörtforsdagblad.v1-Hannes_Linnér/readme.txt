All kod finns i public_html katalogen.

Projektet är uppbyggt med Python ramverket Flask som backend.

Frontend delen består av Html, CSS och Javascript. 
Använder designramverket Bootstrap tillsamman med JavaScriptramverket jQuery.

För att starta projektet behöver man installera alla pluggar till Python.
Sedan starta main filen för att köra igång servern.

För att installera modulerna i Python.
 pip install flask
 pip install flask-login
 pip install psycopg2
